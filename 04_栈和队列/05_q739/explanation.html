LeetCode 739. 可视化解析

单调栈：栈内存放尚未找到更高温的日期下标。当遇到比栈顶高的温度时，依次弹出并计算天数差。

def dailyTemperatures(temperatures):
n = len(temperatures)
ans = [0] * n
stack = []
for i in range(n):
while stack and temperatures[i] > temperatures[stack[-1]]:
prev_index = stack.pop()
ans[prev_index] = i - prev_index
stack.append(i)
return ans
