# LeetCode 84. 柱状图中最大的矩形 (Largest Rectangle in Histogram)

## 题目描述
给定 $n$ 个非负整数，用来表示柱状图中各个柱子的高度。每个柱子彼此相邻，且宽度为 1。
求在该柱状图中，能够勾勒出来的矩形的最大面积。

---

## 1. 暴力解法 (枚举宽度) - $O(n^2)$

### 思路
我们枚举矩形的左边界 $i$ 和右边界 $j$，在 $[i, j]$ 范围内找到矮的那根柱子作为矩形的高度 $h$。
面积 $S = (j - i + 1) \times h$。

### 复杂度分析
- 时间复杂度：$O(n^2)$。我们需要两层循环遍历所有可能的区间。
- 空间复杂度：$O(1)$。

---

## 2. 暴力解法 (中心扩散/枚举高度) - $O(n^2)$

### 思路
既然矩形的高度一定对应某根柱子的高度，我们可以枚举每一根柱子 $i$，以 $heights[i]$ 作为矩形的高度。
我们需要向左和向右找到第一个比 $heights[i]$ 矮的柱子，记为 $left$ 和 $right$。
那么以 $i$ 为高的最大矩形宽度就是 $(right - left - 1)$。

### 复杂度分析
- 时间复杂度：$O(n^2)$。对于每一根柱子，我们都要向两侧搜索。
- 空间复杂度：$O(1)$。

---

## 3. 最优解法：单调栈 (Monotonic Stack) - $O(n)$

### 为什么适合面试？
单调栈是解决“寻找下一个更大/更小元素”这类问题的经典工具。在面试中，如果你能从 $O(n^2)$ 的中心扩散思维进化到 $O(n)$ 的单调栈，能体现出你对数据结构和空间换时间的深刻理解。

### 详细解析 (小白也能懂)

**核心思想**：
在方法 2 中，我们为了找每一根柱子左右两侧第一个比它矮的柱子，做了很多重复的扫描。**单调栈**可以帮我们在一次遍历中记录下这些信息。

**具体步骤**：
1. **栈的定义**：我们维护一个栈，里面存储柱子的**下标**。栈内对应的柱子高度保持**单调递增**。
2. **扫描过程**：
   - 遍历柱子。如果当前柱子高度比栈顶高，说明还没遇到右边界，直接入栈。
   - 如果当前柱子高度比栈顶**矮**，说明栈顶柱子的“右边界”找到了！
   - **出栈并计算**：此时栈顶元素就是我们要计算的“高”。新的栈顶元素就是它的“左边界”。
3. **哨兵技巧**：
   - 在原高度数组前后各加一个高度为 0 的“哨兵”。
   - 前面的 0 可以避免判空，后面的 0 可以保证最后所有柱子都能出栈结算。

**打个比方**：
想象你在排队，你后面来了一个比你矮的人，那么你就“到头了”，可以结算你的矩形面积了。你左边那个比你矮的人，就是你的左起点。

### 复杂度分析
- 时间复杂度：$O(n)$。每个下标仅进栈一次，出栈一次。
- 空间复杂度：$O(n)$。最坏情况下栈的大小为 $n$。
