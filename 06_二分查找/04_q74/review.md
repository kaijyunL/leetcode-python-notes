# LeetCode 74. 搜索二维矩阵

## 题目理解

给你一个 `m x n` 的矩阵 `matrix` 和一个目标值 `target`，判断 `target` 是否在矩阵中。

矩阵满足两个条件：

```text
每一行从左到右升序
每一行的第一个数大于上一行的最后一个数
```

所以整个矩阵可以看成一个被拉平的一维升序数组。

例如：

```text
matrix = [
  [1, 3, 5, 7],
  [10, 11, 16, 20],
  [23, 30, 34, 60]
]
target = 3
```

返回：

```text
True
```

---

## 方法一：线性扫描

### 思路

逐行逐列遍历矩阵，遇到 `target` 就返回 `True`。

遍历完还没找到，就返回 `False`。

### 复杂度

时间复杂度：

```text
O(m * n)
```

空间复杂度：

```text
O(1)
```

## 方法二：展平成一维数组后二分

这是最适合面试的方法。

### 核心思路

虽然 `matrix` 是二维的，但因为它整体有序，可以把它当成一个长度为：

```text
m * n
```

的一维升序数组。

一维下标 `mid` 转回二维坐标：

```python
row = mid // cols
col = mid % cols
```

对应的值是：

```python
matrix[row][col]
```

### 统一模板

继续使用这个模板：

```python
if 满足条件:
    left = mid + 1
else:
    right = mid - 1
```

这题的条件是：

```python
matrix[row][col] < target
```

也就是找：

```text
最后一个小于 target 的位置
```

目标位置就是它右边：

```python
index = right + 1
```

如果 `index` 越界，说明 `target` 比所有数都大。

否则检查：

```python
matrix[index // cols][index % cols] == target
```

### 例子

```text
matrix = [
  [1, 3, 5, 7],
  [10, 11, 16, 20],
  [23, 30, 34, 60]
]
target = 16
```

拉平成一维后相当于：

```text
[1, 3, 5, 7, 10, 11, 16, 20, 23, 30, 34, 60]
```

最后一个小于 `16` 的位置是 `5`，它右边的位置 `6` 正好是 `16`。

所以返回 `True`。

### 面试推荐讲法

1. 线性扫描可以做，但复杂度是 `O(mn)`。
2. 题目保证每行有序，且下一行第一个数大于上一行最后一个数。
3. 所以二维矩阵可以看作一个一维升序数组。
4. 用二分查找最后一个小于 `target` 的位置。
5. 再检查它右边的位置是否等于 `target`。

### 复杂度

时间复杂度：

```text
O(log(m * n))
```

空间复杂度：

```text
O(1)
```
