# LeetCode 34. 在排序数组中查找元素的第一个和最后一个位置

## 题目理解

给你一个升序数组 `nums` 和一个目标值 `target`，返回 `target` 在数组中的起始位置和结束位置。

如果数组中不存在 `target`，返回：

```text
[-1, -1]
```

例如：

```text
nums = [5, 7, 7, 8, 8, 10]
target = 8
```

`8` 第一次出现的位置是 `3`，最后一次出现的位置是 `4`，返回：

```text
[3, 4]
```

---

## 方法一：线性扫描

### 思路

从左到右遍历数组：

1. 第一次遇到 `target` 时，记录左边界
2. 每次遇到 `target`，更新右边界

### 复杂度

时间复杂度：

```text
O(n)
```

空间复杂度：

```text
O(1)
```

## 方法二：二分查找

这是最适合面试的方法。

### 统一模板

继续使用这个模板：

```python
if 满足条件:
    left = mid + 1
else:
    right = mid - 1
```

它的含义是：

```text
找最后一个满足条件的位置
```

---

## 怎么找左边界

左边界是第一个 `target`。

可以转化成：

```text
最后一个小于 target 的位置 + 1
```

所以第一次二分的条件是：

```python
nums[mid] < target
```

找到最后一个 `< target` 的位置后，左边界就是：

```python
start = right + 1
```

---

## 怎么找右边界

右边界是最后一个 `target`。

可以转化成：

```text
最后一个小于等于 target 的位置
```

所以第二次二分的条件是：

```python
nums[mid] <= target
```

找到最后一个 `<= target` 的位置后，右边界就是：

```python
end = right
```

---

## 例子

```text
nums = [5, 7, 7, 8, 8, 10]
target = 8
```

找左边界：

```text
nums[mid] < 8
```

满足条件的是：

```text
5, 7, 7
```

最后一个小于 `8` 的位置是 `2`，所以：

```text
start = 2 + 1 = 3
```

找右边界：

```text
nums[mid] <= 8
```

满足条件的是：

```text
5, 7, 7, 8, 8
```

最后一个小于等于 `8` 的位置是 `4`，所以：

```text
end = 4
```

答案：

```text
[3, 4]
```

## 面试推荐讲法

1. 线性扫描可以做，但复杂度是 `O(n)`。
2. 数组有序，所以用二分把复杂度降到 `O(log n)`。
3. 用统一模板找最后一个满足条件的位置。
4. 左边界 = 最后一个 `< target` 的位置 + 1。
5. 右边界 = 最后一个 `<= target` 的位置。
6. 如果左边界越界，或者 `nums[start] != target`，说明不存在目标值。

### 复杂度

时间复杂度：

```text
O(log n)
```

空间复杂度：

```text
O(1)
```
