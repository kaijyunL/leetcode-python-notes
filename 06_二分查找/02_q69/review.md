# LeetCode 69. x 的平方根

## 题目理解

给你一个非负整数 `x`，返回它的算术平方根的整数部分。

也就是：

```text
返回最大的整数 res，使得 res * res <= x
```

例如：

```text
x = 8
```

真实平方根是：

```text
2.828...
```

只返回整数部分：

```text
2
```

---

## 方法一：线性扫描

### 思路

从 `0` 开始往上试。

找到第一个：

```text
i * i > x
```

那么答案就是：

```text
i - 1
```

### 复杂度

时间复杂度：

```text
O(sqrt(x))
```

空间复杂度：

```text
O(1)
```

## 方法二：二分查找

这是最适合面试的方法。

### 核心思路

这题可以转化成：

```text
在 [0, x] 中，找最后一个 mid，使 mid * mid <= x
```

如果：

```text
mid * mid <= x
```

说明 `mid` 是一个合法答案，但可能还有更大的合法答案，所以继续向右找。

如果：

```text
mid * mid > x
```

说明 `mid` 太大了，要向左收缩。

### 过程

维护：

```text
left = 0
right = x
```

每次取中点：

```python
mid = (left + right) // 2
```

如果：

```python
mid * mid <= x
```

继续向右找：

```python
left = mid + 1
```

否则：

```python
right = mid - 1
```

最后返回：

```python
right
```

### 为什么返回 right

这个模板和第 35 题一致，都是找：

```text
最后一个满足条件的位置
```

第 69 题里，条件是：

```text
mid * mid <= x
```

循环结束后，`right` 就是最后一个平方小于等于 `x` 的位置，也就是平方根的整数部分。

### 面试推荐讲法

面试时可以这样讲：

1. 暴力可以从小到大试，但会慢。
2. 因为平方函数随着数字变大而变大，具有单调性，可以二分。
3. 统一模板：找最后一个满足条件的位置。
4. 这题的条件是 `mid * mid <= x`。
5. 如果满足，就 `left = mid + 1` 继续向右找。
6. 如果不满足，就 `right = mid - 1` 向左收缩。
7. 最后返回 `right`。

### 复杂度

时间复杂度：

```text
O(log x)
```

空间复杂度：

```text
O(1)
```
