# LeetCode 151. 翻转字符串中的单词 (Reverse Words in a String)

## 题目描述
给你一个字符串 `s` ，请你反转字符串中 **单词** 的顺序。
**单词** 是由非空格字符组成的字符串。`s` 中至少包含一个单词。
返回一个由单词组成的字符串，且单词间用单个空格连接，结果字符串不应包含前导空格、尾随空格以及单词间的多余空格。

---

## 解法一：编程语言特性（简单暴力）

这是最直观的方法，利用 Python 内置的强大字符串处理函数：
1. `split()`：默认会按空格切分，并自动处理多余的空格（多个空格按一个处理，忽略首尾空格）。
2. `reversed()` 或 `[::-1]`：翻转切分后的列表。
3. `join()`：将列表中的单词用单个空格连接。

### 算法分析
- **时间复杂度**：O(N)，其中 N 是字符串长度。`split`、`reversed` 和 `join` 都会遍历字符串或列表。
- **空间复杂度**：O(N)，用于存储切分后的单词列表。

---

## 解法二：双端队列 / 模拟（渐进理解）

在不直接使用 `split` 的完整功能时，我们可以手动读取每一个单词：
1. 使用两个指针或循环定位每个单词。
2. 将每个单词插入到结果列表的 **最前面**。
3. 最后合并。

这种方法更接近底层，体现了“翻转”的逻辑（先进后出）。

---

## 解法三：翻转三个阶段（最优解思想）

这是在很多不允许直接使用内置辅助函数的面试中（如 C++ 面试）最经典的 O(1) 额外空间解法（在 Python 中因字符串不可变需 O(N)）。
步骤如下：
1. **清理空格**：移除多余的前导、尾随和中间空格。
2. **整体翻转**：翻转整个字符串。此时单词的顺序对了，但单词内部是乱的。
3. **逐词翻转**：再次翻转每个单词，使其内部顺序恢复正常。

### 示例图解
输入: `"  the sky  is blue  "`
1. 清理空格: `"the sky is blue"`
2. 整体翻转: `"eulb si yks eht"`
3. 逐词翻转: `"blue is sky the"`

### 算法分析
- **时间复杂度**：O(N)。
- **空间复杂度**：O(N)（Python 必须转为 list 才能操作）。

---

## 解法四：基于解法三的 Pythonic 实现

如果我们认同“三步翻转”的逻辑，但想利用 Python 的内置函数来简化代码，可以这样写：
1. `s = " ".join(s.split())` (清理)
2. `s = s[::-1]` (整体翻转)
3. `s = " ".join(word[::-1] for word in s.split())` (单词翻转)

这种写法保留了算法的思维过程，同时极其简洁。

---

## 如何选择？
- **追求速度与简洁**：选解法一。
- **考察数据结构**：选解法二。
- **考察底层字符串操作**：选解法三（面试官最爱）。
